Summary

Supercrypt is a hybridized (Cryptography and Steganography) lossless data security software. It takes any secret or confidential file (mp4, mp3, documents, text) you want to hide from the public, compresses it, encrypts it with a password, and then hides it into an image of your choice. This image will serve as a camouflage to anyone that sees it as they will not know that a piece of information is embedded in the image. This information can be retrieved from the image with the same key used to encrypt it.

The application was built with Python, TKinter, Cryptography, Steganography .
